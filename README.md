# Unprefix Nonce

A WordPress package that wrap the nonce logic in a OOP way.

# Requirements

- PHP 5.6+
- WordPress 4.8+
- [Slugify ^3.0](https://github.com/cocur/slugify)

# Installation

Use [composer](https://getcomposer.org/) 

`composer require unprefix/nonce`
